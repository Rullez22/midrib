# MIDHUB Mobile App (`/app/`) — каркас мобильного приложения

**Дата:** 2026-07-06
**Функция:** Мобильная версия MIDHUB, доступная локально по `http://localhost:3000/app/`.

## Контекст и решение

- **Каркас** («оболочку» апки) берём по аналогии с проектом **Вектор**
  (`/Users/shavkat/Downloads/midhub/source`): splash-экран → мобильный
  шелл с верхним баром и нижней навигацией, роут-флоу.
- **Начинку и стиль** берём из **MIDHUB** (текущий `midhub-app`):
  дизайн-система `src/components/ds/*`, токены `src/tokens/*`, шрифт
  Articulat, light-first.
- **Размещение:** внутри `midhub-app` как сегмент `src/app/app/`.
  Веб-версия MIDHUB остаётся на `/`, мобильная — на `/app/`.
  Так `http://localhost:3000/app/` работает сразу и DS переиспользуется
  напрямую, без копирования.
- **Вид на десктопе:** телефонная рамка (device frame) по центру
  страницы (390×844). На реальном мобильном — полноэкранно.
- В будущем — Telegram Mini App + нативная обёртка iOS/Android. Сейчас —
  только веб-превью по локальной ссылке.

## Архитектура

```
src/app/app/
  layout.tsx          — device-frame + splash-gate + app-shell (chrome)
  page.tsx            — Главная (стартовый экран, пока welcome/placeholder)
  (будущие экраны по мере поступления от пользователя)

src/components/app/
  mobile-frame.tsx    — телефонная рамка (desktop) / fullscreen (mobile)
  splash-gate.tsx     — splash-оверлей при входе (client), затем контент
  app-splash.tsx      — сам splash-экран (лого MIDHUB, анимация)
  app-top-bar.tsx     — верхний бар в стиле MIDHUB (заголовок + back)
  app-bottom-nav.tsx  — нижняя таб-навигация
  app-shell.tsx       — компоновка bar + main(scroll) + bottom-nav
```

## Использованные компоненты DS

- `Tabs`/`Tab` (`ds/tabs`) — табы Веб/Друзья/Работа (variant solid, size m,
  equal; активный перекрашен в green-500 точечным override).
- `Text` (`ds/text`) — вся типографика (h5 заголовок, p2 строки/описания).
- Токены: `--color-green-500`, `--color-primary`, `--color-orange-300`,
  `--color-border`, семантические text-*/bg-surface.
- App-chrome (`components/app/`): `AppHeader`, `AppBottomNav`, `AppShell`,
  `MobileFrame`, `SplashGate`, `app-icons` (инлайн-SVG уровня апки).
