# MIDHUB Mobile — экран «Выберите кейс:» (QR) + кликабельные флоу

**Дата:** 2026-07-07
**Функция:** По нажатию на QR-иконку в шапке табов «Главная» (`/app`) и
«Документы» (`/app/documents`) открывается список кейсов доступа к данным.
Каждый кейс — кликабельный вход в свой флоу.

**Источник:** Figma «QR code» — node `7009:573572`
(файл `8QfpCJU0RUURop4eh5qZ5O`).

## Reuse Analysis (DS-first)

- `AppHeader` (`showBack` + title) — шапка с кнопкой «назад».
- `Text` — типографика строк.
- `ChevronRightIcon` (`app-icons`) — стрелка справа, как в других списках.
- Паттерн строки-списка (`border-b border-border px-4 py-4`) — из
  `access/page.tsx`.
- QR-кнопки уже существовали в `page.tsx` и `documents/page.tsx` —
  только подключили переход (`<button>` → `<Link href="/app/qr">`).

Серый low-fi макет Figma адаптирован под реальный light-first стиль
MIDHUB (белый фон, строки-списка), а не скопирован 1:1.

## Фазы

- [x] Экран `/app/qr` — «Выберите кейс:», 3 кликабельные строки.
- [x] QR-кнопка в `/app` и `/app/documents` → `/app/qr`.
- [x] Заглушка `/app/qr/[case]` (кейсы 2/3) — «Флоу скоро появится».
- [x] **Кейс 1 целиком**: скан → QR → запрос доступа → (Отклонить: сброс /
      Предоставить: строка сервиса в списке «Веб») → карточка сервиса →
      закрыть доступ (диалог → обработка → чат) → строка revoked (урна) →
      «Скрыть сервис»/урна: удаление. Ветка «Редактировать доступ»: диалог →
      поля (кнопка «Предоставить» залочена, разблок при снятии галочки) →
      карточка.
- [x] Список «Веб» переведён на стор (`registrations-store`).
- [x] **Кейс 2 целиком**: скан → QR → запрос доступа ВУЗа (аккордеоны
      Паспорт/Дипломы) → (Отклонить: сброс / Подтвердить: строка университета
      в «Веб» + диплом pending). Оранжевая точка на табе «Документы»; в
      «Выданные» — «Диплом» первой строкой с оранжевыми сепараторами. Клик →
      подтверждение корректности (Отклонить: диплом исчезает / Подтвердить:
      «Поздравляем» → диплом confirmed, полная карточка). Карточка университета
      переиспользует Редактировать/Закрыть доступ из кейса 1.
- [x] **Кейс 3 целиком**: скан → QR → запрос доступа (Паспорт + красный
      баннер + выбор дипломов, «Подтвердить» залочена до выбора) → карточка
      университета (с выбранными дополнениями) → назад → «Диплом» в Документах
      (pending, оранж) → подтверждение внесённых дополнений (3 аккордеона,
      «Диплом магистра 17.06.2020») → «Поздравляем» → готовая карточка.
