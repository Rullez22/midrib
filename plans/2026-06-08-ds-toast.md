# Toast — компонент DS

**Источник:** Figma «UI Контролы» / Toast (node 1229:61463).
**Цель:** уведомление MIDHUB DS — компонент `<Toast>` 1:1 с дизайном.

## Матрица дизайна

- **Variant:** `positive` (зелёный) · `notice` (жёлтый) · `error` (красный) · `info` (синий, белый фон) · `default` (синий, голубой фон)
- **Раскладка:** `title` задан → «2+ строки» (иконка сверху, заголовок 14/22 + тело 12/20, 312px, items-start);
  не задан → «одна строка» (иконка по центру, тело 12/20, 280px, items-center)
- **Крестик:** проп `onClose` (grey-300, hover → dark-800), `closeLabel` для a11y
- Контейнер: border 1px, радиус 4 (`--radius-xs`), padding 8, gap 8.
- Токены фон/рамка/иконка по варианту:
  positive `green-50`/`green-500`, notice `orange-50`/`yellow-500`, error `red-50`/`red-500`,
  info `white`/`blue-midhub-500`, default `blue-midhub-50`/`blue-midhub-500`.
  Текст: заголовок `dark-900`, тело `dark-800`.

## Фазы

### [x] Фаза 1. Компонент и стили
- `src/components/ds/toast.tsx` — `<Toast>` (`forwardRef`, `HTMLAttributes<HTMLDivElement>` без `title`, `role="status"`).
  Пропсы: `variant`, `title`, `onClose`, `closeLabel`, дети — тело. Инлайн-SVG иконки
  `CheckIcon` / `WarningIcon` / `InfoIcon` (16px, цвет варианта) + `CloseIcon`.
- `src/components/ds/toast.css` — классы `.ds-toast*` в `@layer components` на токенах MIDHUB
  (палитра `--color-*`, `--font-sans`/`--weight-*`, `--radius-xs`). Варианты задают `--toast-bg`/`--toast-border`/`--toast-icon`.

### [x] Фаза 2. Подключение в DS
- `src/styles/globals.css` — `@import "../components/ds/toast.css"`.
- `src/components/ds/index.ts` — экспорт `Toast`, типов `ToastProps`, `ToastVariant`.

### [x] Фаза 3. Витрина и проверка
- `src/app/ds/page.tsx` — секция «Toast» (две колонки: «2+ строки» × «одна строка» по всем вариантам).
- `tsc --noEmit` — чисто.

## Итог

Реализован **целиком**. Компонент `<Toast>` со всеми вариантами и двумя раскладками из Figma,
подключён в DS, добавлен в витрину `/ds`. Не осталось ничего.

Использованные компоненты/токены DS: палитра `green/orange/yellow/red/blue-midhub/dark/grey`,
`--font-sans`/`--weight-medium`/`--weight-regular`, `--radius-xs`, утилита `cn`.
Иконки — собственные инлайн-SVG (не remote-ассеты Figma, чтобы не зависеть от 7-дневных ссылок).
