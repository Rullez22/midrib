# Banner — компонент DS

**Источник:** Figma «UI Контролы» / Banner (node 1220:58705).
**Цель:** информационный баннер MIDHUB DS — компонент `<Banner>` 1:1 с дизайном.

## Матрица дизайна

- **Tone (тон):** `info` (синий) · `neutral` (серый, лоадер) · `warning` (оранжевый) ·
  `caution` (жёлтый) · `danger` (красный) · `note` (белый фон, синяя рамка).
- **Раскладка:** иконка (32) + текст (заголовок P2-Medium 16/24 + описание P3 14/22) + кнопка действия (S-32).
- **Кнопка:** `actionLabel` + `onAction` → кнопка в цвете тона; `loading` → кнопка-спиннер (ghost, заблокирована);
  `action` → произвольный слот (перекрывает оба). У `note` кнопка-контур (blue), у `neutral` — лоадер.
- **Ширина:** `width: 100%` контейнера. «Large» (1120) / «Small» (966) из Figma = ширина обёртки витрины,
  а не проп компонента (web-first: баннер тянется на ширину контента).
- Контейнер: border 1px, радиус 4 (`--radius-xs`), padding 16/24, gap 24.
- Токены фон/рамка/иконка/кнопка по тону:
  info `blue-midhub-50`/`-500`, neutral `grey-10`/`grey-200`(S→`grey-100`)/иконка `grey-300`,
  warning `orange-50`/`-500`, caution `yellow-50`/`-500`, danger `red-50`/`-500`/кнопка `red-300`,
  note `white`/`blue-midhub-500`. Текст: заголовок `dark-900`, описание `dark-800`.

## Фазы

### [x] Фаза 1. Иконки
- `src/components/ds/banner-icons.tsx` — 6 инлайн-SVG из Figma-экспорта (route, hourglass, tap, waiting,
  return, information), перекрашены в `currentColor` (тонируются баннером). Return собран из 3 частей
  через `translate` (insets из Figma). Не зависят от 7-дневных remote-ссылок Figma.

### [x] Фаза 2. Компонент и стили
- `src/components/ds/banner.tsx` — `<Banner>` (props: `tone`, `icon`, `title`, children=описание,
  `actionLabel`, `onAction`, `loading`, `action`). Иконка тона по умолчанию, переопределяется через `icon`.
- `src/components/ds/banner.css` — классы `.ds-banner*` в `@layer components` на токенах MIDHUB.
  Тон задаёт `--banner-bg`/`--banner-border`/`--banner-icon`/`--banner-btn-*`.

### [x] Фаза 3. Подключение в DS
- `src/styles/globals.css` — `@import "../components/ds/banner.css"`.
- `src/components/ds/index.ts` — экспорт `Banner`, типов `BannerProps`/`BannerTone`, иконок.

### [x] Фаза 4. Витрина и проверка
- `src/app/ds/banner-demos.tsx` + секция «Banner» в `src/app/ds/page.tsx` (колонки Large × Small по всем тонам).
- `tsc --noEmit` — чисто. Визуальная проверка через Playwright-скриншот — 1:1 с Figma.

## Итог

Реализован **целиком**. Компонент `<Banner>` со всеми тонами и состояниями (кнопка/лоадер/слот) из Figma,
подключён в DS, добавлен в витрину `/ds`. Не осталось ничего.

Использованные компоненты/токены DS: палитра `blue-midhub/grey/orange/yellow/red/dark`,
`--font-sans`/`--weight-medium`/`--weight-regular`, `--radius-xs`, утилита `cn`.
Иконки — собственные инлайн-SVG (currentColor), не remote-ассеты Figma.
