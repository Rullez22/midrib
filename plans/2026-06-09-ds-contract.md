# Создание договора — DS-компоненты

**Источник:** Figma «UI фичи» / создание договора (обзор 164:26629; формы 164:26472,
175:41207, 164:26576, 175:41065; просмотр 164:26630, 183:28, 226:42099; участники+чат
182:48; транзакции 182:19, 228:0, 182:81, 183:27, 622:0, 189:47, 229:0).
**Цель:** создание/просмотр договора + чат + транзакции в DS (правило 1.1).

## Аудит (16 нод)

| Часть | Решение |
|---|---|
| Просмотр договора (заголовок + label/value + бейджи статуса/верификации) | **reuse `MemberCard` + `Badge`** |
| Форма договора (заголовок/исполнитель/номер/код/сумма/комментарий) | **reuse `PropertyForm`** |
| Участники (раскрытие) + чат | **reuse `QuestionCard`** + новый **`MessageComposer`** |
| Транзакции в блокчейне (шапка + строки) | **reuse `TableHeader` + `Item`** |

## Фазы
