# Modal (попапы) — DS-компонент

**Источник:** Figma «UI фичи» / попапы (134:26158/26159/26162, 135:26149, 2099:228640,
2089:225977, 175:41064, 180:112, 206:0, 651:0, 652:0, 584:0/26, 748:0/284, 595:0, 2022:249195).
**Цель:** модальное окно/диалог в DS (правило 1.1). Это недостающий базовый примитив.

## Аудит (18 нод = варианты одного Modal)

Все попапы — один шелл: оверлей + центр. карточка (рамка grey-90, тень) + крестик +
заголовок по центру (H5) + тело + футер кнопок. Контент варьируется:

| Вариант | Контент (reuse) |
|---|---|
| Подтвердить действие / Обратите внимание | текст + `Checkbox` + `Button`×1–2 |
| Удаление | текст + `Button` (secondary / negative) |
| Блокчейн-подтверждение | label/value строки + `Badge` |
| Подтвердить валидатора | `AchievementCard` + `Button` |
| Участники / Роли | список `Item` (+ `Badge`) |
