# Right sidebar setting (Верификации) — DS-компоненты

**Источник:** Figma «UI фичи» / right sidebar — Верификации (обзор 444:80216; 16 нод
состояний: 469:0, 496:59295, 437:15, 443:89194, 611:61819/61934, 445:78439, 498:59311,

**Цель:** панель настроек правого сайдбара в DS (правило 1.1).

## Аудит (16 нод = состояния одной панели)

Все ноды — варианты панели «Верификации»: заголовок + список строк-настроек
(иконка/цвет + переключатель + статус/мета) + кнопка-футер. Разбивка:

| Часть | Решение |
|---|---|
| Контейнер (заголовок + крестик + hairline + футер) | новый **`SidebarPanel`** |
| Строка настройки (бокс 32 цвет/иконка + заголовок + контрол + мета) | новый **`SettingRow`** |
| Переключатели | **reuse `Toggle` / `Checkbox`** |
| Кнопка-футер | **reuse `Button`** (fullWidth) |
| Цвет-бокс (orange/green) | inline (токены `orange-200` / `green-200`) |
