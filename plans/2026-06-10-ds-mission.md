## Аудит — 4 новых композита, остальное reuse

| Часть | Решение |
|---|---|
| Задание / Новость (мета + прогресс + титул + действие) | 🆕 `InfoCard` (один интерфейс на оба) |
| Колонка департамента (цветной хедер + лицо + список отделов) + профиль | 🆕 `ProfileCard` |
| Ценный конечный продукт (аватар + описание + ссылка) | 🆕 `CKPCard` |
| Отдел → Секция → Функция → Технология (каскад + добавить) | 🆕 `OrgColumns` |
| Департамент открт (раскрытие + ЦКП + Подробнее) | `QuestionCard` (reuse) |
| Табы Структура / План развития / Обучение | `Tabs` (reuse) |
| Кнопки / ссылки | `Button` · `Link` |
