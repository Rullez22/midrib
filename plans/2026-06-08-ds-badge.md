# Badge — статус-бейдж / пилюля (компонент DS)

**Источник:** Figma «UI фичи» — статус «Согласован» (`968:90167`, soft green),
код «214» (`672:5788`, solid cyan), «Локальный» (soft orange).
**Цель:** цветной бейдж-пилюля MIDHUB DS — `<Badge>` 1:1 с дизайном.

## Матрица дизайна

- **Форма:** inline-pill, padding 6/16, радиус 4, Articulat Medium 12/20, `white-space: nowrap`.
- **Variant:** `soft` (фон `color-50`, текст `color-400`) · `solid` (фон `color-200`, текст white).
- **Color:** green · orange · red · blue · cyan · yellow · purple · grey (токены `*-50/200/400`).
- Соответствие макету: «Согласован» = soft green (`green-50`/`green-400`);
  «Локальный» = soft orange; «214» = solid cyan (`cyan-200` + white).
