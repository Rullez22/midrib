## Аудит — reuse + 8 новых композитов

**Переиспользовано:** `Tabs` (Tabs midhub), `TableHeader`, `Item`, `Badge` (solid —
цветные коды 214/216/215 и счета), `Combobox` (Все коды), `Button` (Показать ещё /
Создать отчёт), `QuestionCard` (сворачиваемая характеристика), `Link` (ссылки в
строках), токены MIDHUB.

**Новые композиты (`src/components/ds/composite/`):**

| Композит | Покрывает ноды | Из чего собран |
|---|---|---|
| `StatSummary` | Header information ×6, Label | бордер-карточка с колонками label+value |
| `ReportPeriodBar` | header date & history ×2 | период + иконка календаря + история |
| `TransactionsTable` | транзакции-* ×9 | панель фильтров + `TableHeader` + строки (`Badge`/`Link`) + `Button` |
| `LineChart` | Graph partner | SVG: сглаженная линия + сетка + оси + тултип |
| `AccountCharacteristics` | table 675:64420 | `QuestionCard` + строки-определения (pair/inline) |
| `ArticlesTable` | Статьи, Статьи расходов, footer для статьи | `TableHeader` + строки + `Link` + строка «Итого» |
| `IncomeSources` | Источники поступлений | `Badge` (счёт) + описание + код + значение |
| `ReportFooter` | footer 684:16 | период + сумма + `Button` «Создать отчёт» |

Цветные чекбоксы-фильтры кодов — инлайн-контрол внутри `TransactionsTable`
(DS `Checkbox` монохромный). Mini-варианты нод — те же композиты в меньшей
ширине (web-first адаптив через flex).
