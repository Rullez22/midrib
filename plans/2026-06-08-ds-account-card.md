# AccountCard — карточка счёта (композит DS)

**Источник:** Figma «UI фичи» / карточка счета раскрыта (node `2237:231380`).
**Цель:** карточка счёта кооператива — композит `<AccountCard>` 1:1 с дизайном.

Композит в `src/components/ds/composite/` (см. правило 9 в `docs/MIDHUB_WORKFLOW_RULES.md`).

## Матрица дизайна

- **Баланс** (h144): по центру «Баланс» (16/24 dark-900) · сумма (Medium 32/40 dark-900) ·
  вторичная (16/24 dark-800); по бокам — кнопки `leftAction`/`rightAction` (DS Button m,
  «Реквизиты» ↗ / «Перевод» ↘), gap 72.
- **Панель характеристик:**
  - Шапка (h48, `grey-10`, border-bottom `grey-90`): заголовок по центру (Medium 12/20) +
    крестик (16px, `grey-300`) справа.
  - Строки (12/20, padding 16/24, разделитель `grey-90`): лейбл `grey-300` слева (260px) +
    значение `dark-900` справа (многострочное — список ОКВЭД / распределение).
  - Парная строка (`secondary`): две stacked-ячейки (label+value) + вертикальный разделитель.
- **Футер:** кнопка во всю ширину — белый фон, верхняя граница `blue-500`, текст `blue-500`
  Medium 16/24, скругление снизу.

## Использованные DS-компоненты

- `<Button size="m">` — кнопки «Реквизиты»/«Перевод» (передаются в слоты).
- Токены MIDHUB: `dark-800/900`, `grey-10/90/300`, `blue-midhub-500`, `white`,
  `--radius-xs`, `--font-sans`, `--weight-regular/medium`. Иконки ↗/↘/× — локальные SVG.
